# Legal Document Redaction & NER
This repository focuses on Named Entity Recognition (NER) for legal documents.

## Dataset
We are using the [Text Anonymization Benchmark (TAB)](https://huggingface.co/datasets/ildpil/text-anonymization-benchmark) which contains 1,268 ECHR court cases annotated for anonymization.

### How to use
The data is stored in the `data/` directory in JSON Lines format.
